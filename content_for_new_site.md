# IMPulse - ChatOps Incident Management Platform

_Fast. Simple. Open Source_

**Decsription:**
- IMPulse is a single component, configured with one file
- Fully open source, self-hosted

**Features:**
Free:
- Unlimited escallation policies
- Unlimited oncall schedules
- AI integration
- Custom incident message
- Any external integrations
- Sign in using your messenger
- Supress related incidents

Paid:
- Analytics
- Conditional escallation policies
